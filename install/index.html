Install

Requirements

  • Python 3.4+
  • gmusicapi
  • gmusicapi-wrapper
  • mutagen
  • docopt
  • ffmpeg or avconv for uploading (See here)

Installation

  • Stable release

    pip install gmusicapi-scripts
pip install git+https://github.com/thebigmunch/gmusicapi-scripts

  • Development release

    pip install git+https://github.com/thebigmunch/gmusicapi-scripts@devel
